Git Bash CheatSheet

Tabla de comandos.

Comando Funcionalidad
git init Inicializa un repositorio git en la carpeta actual.
git add Añade archivos al área de preparación.
git commit Guarda los cambios en el repositorio.
git push Sube los cambios al repositorio remoto.
git pull Descarga los cambios del repositorio remoto.

En esta tabla podemos observar distintos comandos y sus funcionalidades.
