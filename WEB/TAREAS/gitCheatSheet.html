Git Bash CheatSheet

Git Bash es una terminal que permite usar Git en sistemas Windows. Proporciona un entorno similar a Linux, donde puedes ejecutar comandos de Git y otros comandos de Unix como ls, cd, rm, cat, etc.

¿Para qué sirve?

  • Usar comandos de Git en Windows sin necesidad de instalar un terminal de Linux
  • Ejecutar scripts de Bash en Windows
  • Acceder a herramientas de línea de comandos como ssh, scp, grep, etc

Tabla de comandos.

Comando Funcionalidad
pwd (Print Working Direct) Sirve para ver la dirección de carpeta en la que te ecuentras
cd (Change Direct) Sirve para cambiar de dirección a la que necesites.
ls (List) Muestra todos los archivos disponibles en la ubicación que te encuentres.
touch Para crear un nuevo archivo en donde sea que te encuentres.
rm Para borrar un archivo.
