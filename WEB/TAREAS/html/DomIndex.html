Preguntas

  1. ¿Qué es el DOM?

    2. El DOM (Document Object Model) es una representación estructurada del documento HTML o XML que permite a los lenguajes de programación manipular su contenido, estructura y estilo.

  2. ¿Qué es un nodo?

    3. Un nodo es una unidad básica del DOM que puede representar un elemento, un atributo, un texto, un comentario, entre otros.

  3. ¿Cómo se accede a un elemento del DOM?

    4. Se puede acceder a un elemento del DOM utilizando métodos como getElementById, getElementsByClassName, getElementsByTagName, o querySelector.

  4. ¿Cómo se modifica un elemento del DOM?

    5. Un elemento del DOM se puede modificar cambiando sus propiedades, como innerHTML, style, o utilizando métodos como setAttribute.

  5. ¿Cuál es la diferencia entre getElementby* y querySelector*?

    6. getElementBy* devuelve elementos específicos (por ID, clase, o etiqueta), mientras que querySelector* utiliza selectores CSS y puede devolver el primer elemento que coincida o una lista de elementos.

  6. ¿Qué es un evento?

    7. Un evento es una interacción o cambio que ocurre en el navegador, como un clic, un movimiento del mouse, o la carga de una página.

  7. ¿Qué es un listener?

    8. Un listener es una función que "escucha" un evento y se ejecuta cuando dicho evento ocurre.

  8. ¿Qué es un callback?

    9. Un callback es una función que se pasa como argumento a otra función y se ejecuta después de que ocurre un evento o se completa una operación.

  9. ¿Qué es un objeto de evento?

    10. Un objeto de evento contiene información sobre un evento específico, como el tipo de evento, el elemento objetivo, y otros detalles relacionados.
