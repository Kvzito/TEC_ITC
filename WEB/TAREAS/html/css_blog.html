CSS Basics

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para definir la apariencia y el diseño de las páginas web. CSS controla aspectos visuales como el color, la tipografía, el espaciado, la disposición y muchos otros aspectos de los elementos HTML en una página.

Selectores

Los selectores son patrones que se utilizan para seleccionar los elementos a los que se les aplicarán las reglas de estilo. Los selectores más comunes y sus diferencias son:

  • Selector de tipo: Se aplica a todos los elementos de un mismo tipo, sin importar sus clases o ID's.
  • Selector de clase: Aplica a todos los elemntos de una clase en específico, pudiendo aplicarse así a múltiples elementos sin afectar a todos los de un mismo tipo.
  • Selector de ID: Aplica a un único elemento ya que este identificador es único por lo que es irrepetible en el documento.

Propiedades

Las propiedades en CSS definen los aspectos específicos que se quieren modificar de un elemento. Por ejemplo, si se quiere cambiar el color de fondo de una página o el tamaño de una fuente, esas modificaciones se hacen a través de propiedades como background-color o font-size.

Valores

Los valores son los datos que se asignan a las propiedades. Por ejemplo, en color: red;, "color" es la propiedad y "red" es el valor. Los valores pueden ser números, cadenas de texto, colores, unidades de medida, entre otros.
